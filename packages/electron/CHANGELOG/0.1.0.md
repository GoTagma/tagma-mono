The first public alpha of Tagma Desktop. An Electron shell wraps the editor
with a desktop-native window, the bundled editor sidecar, and per-workspace
single-instance enforcement.

### Added

- Electron main process with editor bridge and single-instance-per-workspace
  enforcement
- Bundled editor sidecar launched as a child process
- Platform installers: Windows NSIS, Linux AppImage, macOS DMG (Intel + Apple
  Silicon)

### Known limitations

- Installers are not code-signed yet; macOS will show "damaged / cannot be
  opened" until right-click → Open, Windows will show a SmartScreen warning.
  Code signing will be wired once certificates are in place.
- No in-app auto-updater yet. Update feeds (`latest.yml`, `latest-mac.yml`)
  are uploaded so updater support can be turned on without breaking this
  install.
